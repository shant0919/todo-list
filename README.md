# To-Do List

A simple to-do list web app built with HTML, CSS, and JavaScript.

### Features

- Add tasks
- Mark tasks as complete
- Delete tasks
- Tasks are saved in the browser (localStorage), so they stay after refreshing

### Built With

- HTML
- CSS
- JavaScript

### How to Run

Clone the repo and open `index.html` in your browser — no setup needed.

```bash
git clone https://github.com/shant0919/todo-list.git
```
